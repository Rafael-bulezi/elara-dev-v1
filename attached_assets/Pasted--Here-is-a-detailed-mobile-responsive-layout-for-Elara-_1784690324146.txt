### Key Mobile Behaviors & Responsive Rules

| Element                  | Mobile Behavior                                      | Reason |
|--------------------------|------------------------------------------------------|--------|
| **Top Navigation**       | Logo + icons only, search becomes icon that expands | Saves vertical space |
| **Hero Banner**          | Full width, taller image, big CTA button            | Grabs attention immediately |
| **Categories**           | Horizontal scroll with icons + labels                | Easy one-handed browsing |
| **Product Cards**        | Vertical stack, larger images, fewer per row         | Better readability on small screens |
| **Carousels**            | Horizontal scroll with snap, visible partial cards   | Encourages exploration |
| **Mixeiro Section**      | Prominent call-to-action card                        | Highlight local feature |
| **Trust Badges**         | Compact row of icons                                 | Reassurance before checkout |
| **Bottom Navigation**    | Fixed bottom bar (Home, Categories, Chat, Account)   | Fast access on mobile |

### Additional Responsive Details

- **Font sizes**: Bigger on mobile for readability
- **Spacing**: Generous padding/touch targets (minimum 48px)
- **Images**: Use responsive sizes (smaller on mobile)
- **Navigation**: Hamburger menu for full categories if needed
- **Scroll behavior**: Smooth, with lazy loading for performance
- **Safe areas**: Respect iOS/Android notches and bottom gestures
